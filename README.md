**Modules:**

Utility - Utility Commands

Mod - Moderation Commands

Info - Info Commands

Music - Music Commands

Other - Other Commands

```
Syntax for Sirk:
<input> - Input for the Command
[o] - Optional
"|" - Alias
(permission) - Permission
```
